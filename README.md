# Replication Package

This repository provides the review data and supporting spreadsheets used in the study selection, quality assessment, and data extraction process for the paper:

**Integrating LLM Agents with Heterogeneous Systems via the Web of Things**

## Overview

The repository contains the primary research artifacts used to manage and document the literature review workflow, including:

- raw bibliographic exports
- screening decisions,
- full-text inclusion/exclusion decisions,
- quality assessment results,
- structured data extraction

## Files

| File | Description |
|------|-------------|
| `Screening.xlsx` | Title/abstract screening decisions from reviewers |
| `review_full_text_decision.xlsx` | Final full-text screening decisions and resolution notes |
| `Quality_assessment.xlsx` | Individual and final quality assessment results |
| `Data_Extraction.xlsx` | Final structured extraction table for included studies |
| `export2025.11.17-03.23.40.csv` | IEEE exported bibliographic/search records |
| `scopus_export_Nov 17-2025_*.csv` | Scopus exported bibliographic/search records |
| `acm.txt` | ACM exported bibliographic/search records |
